# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability in Peanut Butter AND JAM (PBNJ), please report it responsibly.

**Do not** open a public GitHub issue for security-sensitive bugs.

- **Preferred**: Email the maintainers (e.g. via the contact details on [peanutbutterandjam.xyz](https://www.peanutbutterandjam.xyz/get-in-touch)) with a clear description of the issue and steps to reproduce.
- Alternatively, use [GitHub Security Advisories](https://github.com/Esscrypt/peanutbutterandjam/security/advisories/new) for this repository if you have a GitHub account.

We will acknowledge your report and work with you to understand and address it. We ask for reasonable time to fix the issue before any public disclosure.

## Supported versions

Security updates are applied to the current main branch and the latest release(s). We recommend running a recent release or main and upgrading when security fixes are announced.
